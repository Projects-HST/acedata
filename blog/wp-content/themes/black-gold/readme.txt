-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
       Frequently Asked Questions
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-


1.  How do I add widgets to my home sidebar/footer ?

    -   From the WordPress admin sidebar choose "Appearance"
    -   Select "Customize"
    -   You will now be presented with this theme's customizer screen, click on "Widgets"
    -   You will now have the choice between adding widgets to the home sidebar or footer
    -   After selecting one you will be presented with the "Add a Widget" button, select this to add widgets
    -   You will see a list of widgets you can then add, you may also wish to search for particular 
        widgets using the search box. 
    -   Clicking on a widget will give you a preview of how that widget will appear on the screen
    
    ! Note  
        - Home sidebar widgets will either appear on the left or the right of the screen
          depending on which setting you've chosen.
        - Footer widgets will always appear at the bottom of the screen.


2. How do I change the positioning of my Home Sidebar?

    -   From the WordPress admin sidebar choose "Appearance"
    -   Select "Customize"
    -   You will now be presented with this theme's customizer screen
    -   Select "Home Sidebar Layout"
    -   You will now see a list allowing you to position your home sidebar either to the right or the left

3. How do I set my homepage so that it doesn't display a list of posts?

    -   From the WordPress admin sidebar choose "Settings"
    -   Select "Reading"
    -   You will then see a label like this "Front page displays"
    -   To the right of this will be two options, "Your latest posts" & "A static page (select below)"
    -   Select "A static page"
    -   You can then select which page you wish to be your homepage by selecting
        your desired homepage from the dropdown list beside "Front page:"
    -   You can then select which page you wish to be your blog page by selecting
        the desired blog page from the dropdown list beside "Posts page:"
